# 2026-03-30 작업 요약

## 오늘 반영한 주요 변경

### 1. 웹서비스 구조 및 보안 보강

- 작업 접근을 `job_id + edit_token` 기반으로 분리해서, 일반 사용자 흐름에서는 다른 사용자의 작업에 관여하지 않도록 구조를 바꿈
- 공용 작업 목록 API 제거
- `Turnstile` 검증 추가
- 서버 측 업로드 rate limit 추가
- 완료/실패 작업 TTL 자동 정리 추가
- 업로드를 파일 전체 메모리 적재 방식이 아니라 스트리밍 저장 방식으로 변경
- 공개 웹서비스 기준으로 CORS 기본값을 닫힌 상태로 변경

### 2. 변환 처리 성능 개선

- 변환 엔진 호출을 subprocess 기반으로 분리
- 워커 동시 처리 옵션 추가
- 운영 설정은 `concurrency 3`으로 반영
- 동시 업로드 테스트 결과, 현재 환경에서 안전하게 병렬 처리 가능한 수준으로 확인

### 3. 공개 배포 구성 정리

- WSL + Docker Compose + Cloudflare Tunnel 기반 배포 구성 정리
- `systemd` 자동 시작 서비스 반영 및 검증
- GitHub Pages 프론트 배포 워크플로 유지
- 공개 백엔드 주소를 `https://api.govpress.cloud` 기준으로 운용

### 4. 데스크톱 웹 UI/UX 정리

- 데스크톱은 편집기/미리보기 병렬 작업형으로 정리
- 툴바 중심 구조로 단순화
- 최근 작업 공용 목록 제거
- 정보 모달 문구와 대기화면 설명 정리
- 다크모드, Typography, 문단 강조, 표 렌더링 등 세부 다듬음

### 5. 모바일/태블릿 UI 분기

- 모바일은 리더형, 태블릿은 하이브리드 상하 분할, 데스크톱은 좌우 병렬로 분기
- 모바일/태블릿에서는 `미리보기 / 편집` 상단 탭 중심 구조로 정리
- 모바일 하단 저장 버튼 제거
- 모바일에서 `목록으로` 버튼 제거
- 모바일에서는 다크모드 버튼 제거
- 모바일/태블릿에서도 복사/공유 버튼은 유지

### 6. 웹 기본 설정 보정

- 웹에서는 저장된 예전 API 키를 무시하고 현재 기본 공개 키만 사용
- 웹 기본 서버 주소를 항상 `https://api.govpress.cloud`로 고정
- 이 수정은 `github.io` 배포에도 반영되도록 커밋 및 푸시 완료
