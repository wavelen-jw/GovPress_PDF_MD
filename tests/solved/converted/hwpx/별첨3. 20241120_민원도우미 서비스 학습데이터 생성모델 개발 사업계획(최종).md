# 민원도우미 서비스 학습데이터 생성 모델 개발 사업계획

### 개 요

- (사 업 명) 민원도우미 서비스 학습데이터 생성 모델 개발
- (사업기간/예산) ’24. 12. ~ ’25. 3. (3개월) / 402,000천원
- (사업목적) 생성형 AI기반 민원상담 관련 서비스 개발에 필수적인 학습데이터를 자동 생성하는 모델을 개발하여 민원도우미 서비스*의 신속한 개발 지원
  > 콜센터 직원의 민원 답변 지원, 챗봇을 통한 온라인 민원응대 등

### 사업내용

- 공공분야 민원상담 원천데이터* 수집‧분석
  - 데이터의 수집 가능성, 규모, 가공 용이성, 학습 효과성 등을 고려하여 범용적으로 활용 가능한 민원상담 관련 원천데이터 수집‧분석
    > 정부민원안내콜센터(110), 정부24 등 상담(녹취)데이터, 국민비서 민원상담(58종) 데이터 등
- 학습데이터를 자동으로 생성하는 AI 모델 개발용 데이터 구축
  - 수집한 원천 데이터를 활용하여 개인정보 제거 및 음성-텍스트 변환* 등 전처리 * STT(Speech To Text)
  - 전처리된 데이터로 민원도우미 서비스 개발용 학습데이터*를 쉽게 생성해주는 모델 개발에 필요한 데이터 구축 * 질의–응답 형식
- 학습데이터를 자동으로 생성하는 AI 모델 개발‧검증
  - 구축한 데이터를 상용 AI 모델에 학습시켜 모델 개발 및 자동 생성된 학습데이터의 적절성 여부를 평가하여 모델 성능 검증

### 모델 활용

- 범정부 생성형AI 공통기반(’25년 구축 예정)에 등록·공유하여 민원상담 도우미 서비스* 개발에 활용
  > (110콜센터) 콜센터 상담프로그램에 생성형 AI 기반 도우미 제공,
