# 「범정부 소버린 AI 공통기반」설명자료

### 목표

- (공공기관) AI 서비스를 신속·안전하게 도입할 수 있는 기반 제공
- (민간기업) 업무망에서, 공공기관 내부 데이터를 활용하는 AI 를 구독형으로 제공할 수 있는 환경 제공

### AI 공통기반 구성 및 역할

| --- |

- (공통기반) 인프라 + AI 플랫폼 + 지원기능
  - 인프라 : 업무망에서 내부데이터 이용 가능한 국가정보자원관리원 대구센터 민관협력(PPP) 존 內 GPUaaS* 활용
    > SDS, NHN, KT 제공(‘25년) / 국가AI컴퓨팅센터 구축 후 적용 예정(’26~)
  - AI플랫폼 : 파운데이션 모델, AI서비스 개발을 위한 지원도구
    > (확장성) 신규 플랫폼·서비스는 국정원보안성검토 후 제약 없이 PPP존 입주,
    > (유연성) SW구매가 아닌 구독형으로 운영하여, MCP, AI agent, 멀티모달,
  - 지원기능 : 공통 RAG, AI서비스마켓*, AI학습용 데이터, 사용자 관리, 미터링 등 다수 AI플랫폼·서비스가 필요로 하는 기능 제공
    > 민간 SaaS 서비스 목록·설명페이지, 체험, 조달(디지털서비스몰)과 연계하여 구독까지 지원
- (AI서비스) 다수기관 공통서비스는 민간 SaaS 서비스 구독하고,
  - 공통서비스 구독 : 다수기관에서 활용되는 SaaS 서비스*는 행안부 일괄구매-배포 방식이 아닌 개별기관이 서비스를 구독하여 활용
    > (예) ’지능형 업무관리 플랫폼‘의 소통·협업 도구(BrityWorks, 네이버웍스)
  - 특화서비스 구축 : 구현부담을 줄이고 업무와 일하는 방식의 AI전환에 집중할 수 있도록 공통기반에서 지원하는 AI플랫폼 활용
- (AI도입지원) 공통기반 SaaS 서비스 확대, AI전환 전문인력 지원*, AI 서비스 도입을 위한 가이드라인 배포 등 지원프로그램 운영
  > (사례) 팔란티어 FDE(Forward Deployed Engineer) 프로그램은 엔지니어를 고객사에 배치하여 데이터 통합, AI기반 워크플로우 등을 개발하여 AI전환 가속화

### 공공AX 전환을 위한 MCP(Model Context Protocol) 확산

  > 인공지능 에이전트가 외부 데이터나 서비스에 접근하는 인터페이스 표준(앤트로픽社 발표)
- (방안) 공통기반에 탑재되는 AI서비스가 정부 내 데이터, 서비스API 등 리소스에 접근할 수 있도록 MCP Host를 공통기능으로 규정*
  > 프로그램을 직접 구축하는 것이 아닌, AI 플랫폼이 기능을 제공할 수 있도록 요구사항으로 관리
  - MCP 서버 구축이 필요한 대상(서비스, DB, 파일 등) 식별 및 목록화
    > (파일) ‘지능형 업무관리 플랫폼’ 도입을 통해 개인PC의 문서를 클라우드로 이동
    > (서비스) 업무망은 직접연계, 인터넷망 서비스는 국정원 N2SF실증을 거쳐 연계 추진
    > (DB) 보안이 필요한 주요 데이터는 대다수 업무망에 위치하므로 직접 연계
  - 인증‧보안정책 적용, 도구 정의, 요청‧응답 로깅 등 요구사항 규정
- (역할) 행안부는 가이드라인 수립, 예산지원(‘26년 RAG 구축예산00억원 활용) 등 MCP 확산 지원, 대화형 AI서비스와 연계한 MCP 활용 선도사례 창출
  - 각 부처는 소관 리소스에 MCP 서버 설치, 접근권한 설정

| --- |

- MCP(Model Context Protocol) 개념
  - 엔트로픽社가 발표한 인공지능 에이전트의 외부 데이터/서비스
    > MCP Host : AI서비스로서, LLM 모델의 요청을 받아 MCP 서버와 통신하는 브로커
    > MCP Client : LLM 모델 내부의 기능으로, 외부 접근이 필요할 때 Host에 요청
    > MCP Server : AI서비스의 요청을 받아 작업을 수행하고 결과를 반환
- 공공 분야 MCP 활용
  - 각 부처 보고서 작성 시, AI가 데이터베이스 내 통계‧원본 데이터를 직접 가져와 분석 가능
    > 예: "작년 월간 인구 변화, 인구 유입량 데이터를 분석해줘" 명령 시 AI가 MCP 통해 데이터 분석
  - 개인 비서 에이전트가 캘린더 앱, 예약 사이트, 교통편 등 MCP 설치된 외부 서비스에 접속하여 업무 수행 가능
- 공공 AX 프로젝트에서의 MCP 중요성
  - 공공 AX 프로젝트의 최우선 과제는 공공 내부 데이터베이스, 문서 아카이브 등에 MCP 설치하는 것
  - 각 부처는 데이터, 리소스, API가 있는 곳에 MCP 적극 설치 필요하며, 이후 새롭게 출시되는 AI모델을 적용하는 순서로 확대
  - 또한, MCP가 로봇, 드론, UAM 등에 탑재 시 피지컬 AI로 발전 가능
