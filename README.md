# 정부 보도자료.Markdown

정부 보도자료 `PDF`와 `HWPX`를 사람이 읽기 쉬운 Markdown 문서로 바꾸고, 바로 수정하고 저장할 수 있는 웹서비스입니다.

서비스 주소:
- `https://wavelen-jw.github.io/GovPress_PDF_MD`

## 할 수 있는 일

- 정부 보도자료 PDF/HWPX를 Markdown으로 변환
- Markdown 문서를 바로 열어 편집
- 변환 결과를 미리보기로 확인
- HWPX 표를 `표: MD` 또는 `표: HTML` 기준으로 확인
- 필요한 표현을 직접 수정
- Markdown 파일로 저장
- 복사하거나 공유
- `참고` 부록, 담당부서, 보도자료형 bullet/소제목 구조를 최대한 유지
- 문제 파일(`tests/problem/`) 기반으로 변환 규칙을 계속 개선

## 사용하는 방법

1. `파일 열기`를 눌러 PDF, HWPX 또는 Markdown 파일을 엽니다.
2. PDF/HWPX를 열면 자동으로 Markdown으로 변환됩니다.
3. HWPX는 먼저 `표: MD` 결과가 표시되고, `표: HTML`은 준비되면 선택할 수 있습니다.
4. 화면 오른쪽 또는 아래의 미리보기에서 결과를 확인합니다.
5. 필요한 문장을 바로 수정합니다.
6. `저장하기`로 Markdown 파일을 내려받습니다.

## Markdown이란?

Markdown은 제목, 목록, 강조 같은 문서 구조를 간단한 기호로 적는 방식입니다.

예:
- `# 제목`
- `- 목록`
- `**강조**`

별도 전용 뷰어가 없어도 웹브라우저나 일반 문서 편집기에서 읽기 쉽고, AI가 문서 구조를 이해하고 활용하기에도 적합합니다.

## 이런 문서에 잘 맞습니다

- 정부 부처 보도자료
- HWPX 원본으로 배포되는 행정기관 보도자료
- 기관 공지문
- 제목, 문단, 목록 중심 문서
- PDF를 Markdown 초안으로 빠르게 바꾸고 싶은 문서

## 이런 경우는 직접 확인이 필요합니다

- 복잡한 표가 많은 문서
- 그림, 도표, 캡션이 많은 문서
- 참고자료, 주석, 붙임문서, 요약표가 복잡한 문서
- 원본 서식을 그대로 유지해야 하는 문서

변환 결과는 초안입니다. 공개 전에는 제목, 표, 목록, 링크, 숫자 표기를 한 번 더 확인하는 것을 권장합니다.
특히 HWPX의 복잡한 표는 `표: MD`와 `표: HTML`을 비교해 더 적절한 쪽을 선택하는 것이 좋습니다.

## 저장과 공유

- `저장하기`: 현재 편집 내용을 Markdown 파일로 저장합니다.
- `복사`: 현재 내용을 클립보드에 복사합니다.
- `공유`: 기기나 브라우저에서 지원하는 방식으로 내용을 공유합니다.

## 참고

- 이 서비스는 `GitHub Pages 프론트 + FastAPI API + worker` 구조로 동작합니다.
- 공개 프론트는 `https://wavelen-jw.github.io/GovPress_PDF_MD`입니다.
- 웹 첫 접속 기본 서버는 `서버W`이며, 정보 메뉴에서 `서버V`/`서버W`/`서버H` 서버를 선택할 수 있습니다.
- 공개 웹에서는 저장된 개인 API 키 대신 공개용 기본 API 키를 사용합니다.
- 이 서비스는 여러 사람이 동시에 접속할 수 있지만, 각 작업은 개별적으로 처리됩니다.
- 업로드한 원본 파일과 작업 결과는 일정 시간이 지나면 자동 정리됩니다.

## 개발 메모

- 현재 변환 규칙은 PDF뿐 아니라 HWPX 보도자료에도 맞춰 계속 보정 중입니다.
- 품질 개선은 `tests/problem/`의 문제 파일과 정답 `.md`를 기준으로 반복합니다.
- 자세한 규칙은 `docs/conversion-rules.md`를 참고합니다.
