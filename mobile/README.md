# GovPress Mobile

Expo 기반 모바일 MVP 클라이언트입니다.

## 기능

- PDF 문서 선택
- GovPress 백엔드 업로드
- 최근 작업 목록 조회
- 작업 상태 확인
- Markdown 결과 보기
- Markdown 수정 저장
- Markdown 파일 공유
- 서버 주소 / API 키 설정

## 실행

```bash
cd mobile
npm install
npx expo start
```

기본 백엔드 주소는 `http://127.0.0.1:8012`입니다.
실기기에서는 모바일 기기에서 접근 가능한 PC IP 주소로 바꿔야 합니다.
