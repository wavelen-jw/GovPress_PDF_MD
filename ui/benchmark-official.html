엔진별 전체 점수

아래 표는 로컬 clone의 opendataloader-bench 공식 코퍼스와 evaluation.json을 기준으로 다시 정리한 리더보드입니다. 읽힘은 이번에 별도 엔진으로 추가해 같은 ground truth와 같은 evaluator로 평가했습니다.
정확도 지표인 Overall / NID / TEDS / MHS는 같은 코퍼스와 같은 evaluator 위에서 직접 비교 가능합니다. 다만 속도는 엔진마다 실행 시점과 하드웨어가 달라 이 페이지에서는 참고값으로만 보여줍니다.

방법과 출처

      

        이 페이지는 OpenDataLoader 공식 벤치 방식을 그대로 따르되, 여기에 읽힘을 하나의 PDF 변환 엔진으로 추가한 결과를 정리한 것입니다.
        출처 표시는 아래 세 축으로 나눴습니다. 공식 지표 설명, 공식 코퍼스와 evaluator 구현, 그리고 이번에 추가한 읽힘 실행 방식입니다.
      

      

    


    

      
지표 해설

      

        지표 정의와 수식의 원문 설명은 OpenDataLoader 공식 문서를 기준으로 삼았습니다.
        여기서는 페이지 해석에 필요한 수준으로만 요약하고, 자세한 근거는 공식 문서 링크를 함께 남깁니다.
